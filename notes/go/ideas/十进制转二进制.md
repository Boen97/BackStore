# 十进制转二进制

> 除2取余，逆序排列,直到商为 0 为止

## 十进制整数转二进制整数，example 十进制 173 转二进制

1. 173 / 2 = 86, 173 % 2 = 1
2. 86 / 2 = 43, 86 % 2 = 0
3. 43 / 2 = 21, 43 % 2 = 1
4. 21 / 2 = 10, 21 % 2 = 1
5. 10 / 2 = 5, 10 % 2 = 0
6. 5 / 2 = 2, 5 % 2 = 1
7. 2 / 2 = 1, 2 % 2 = 0
8. 1 / 2 = 0, 1 % 2 = 1

- 逆序排列得 10101101

## 十进制小数转二进制小数

> 乘2取整，顺序排列,直到积中的小数部分为 0, 每次用剩余下的小数部分

- 0.8125
1. 0.8125 * 2 = 1.6250, 整数部分为 1
2. 0.6250 * 2 = 1.2500, 整数部分为 1
3. 0.2500 * 2 = 0.5000， 整数部分为 0
4. 0.5000 * 2 = 1.0, 整数部分为 1

- 顺序排列 为 （0.1101）
